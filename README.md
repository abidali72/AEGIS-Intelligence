# AEGIS Intelligence v5.0

Aegis Intelligence is a Biometric & Security Platform using YOLOv8 for real-time object and behavior detection (such as smoking detection). Built with Python and `customtkinter`.

## Features
- Real-time biometric analysis and motion tracking
- Smoking detection using YOLOv8
- Automated event logging with SQLite
- Modern UI built with `customtkinter`
- Contextual event analysis and history tracking

## Setup
1. Create a virtual environment and install dependencies (e.g., `opencv-python`, `customtkinter`, `ultralytics`, `Pillow`).
2. Run the application:
   ```bash
   python app.py
   ```
